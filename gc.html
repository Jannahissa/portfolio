Research & Innovation Project ✨

GestureCraft

Transforming Education through Gestural Interactions ★

Bridging Play and Learning ★

I created a groundbreaking educational experience by taking Minecraft and infusing it with the magic of gestures. Children can now navigate and build within the virtual realm using their own hands, offering an unparalleled hands-on learning experience.

Key Technologies ★
  • User Research
  • Interaction Design
  • Gesture Recognition
  • Minecraft API
  • Python
  • OpenCV
Research Methods ✨
  • Conducted user study with 15 participants (ages 6-17)
  • A/B testing for gesture instruction methods
  • Spatial reasoning tasks with 10 participants
  • Real-time gesture recognition integration
Key Findings ★
  • Enhanced viewfinder and object manipulation
  • Improved spatial cognition with embodied gestures
  • Better memory recall through gesture-based interactions
  • Successful integration with Minecraft API
