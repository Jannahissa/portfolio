Split: Revolutionizing Bill Sharing ★

Split is an innovative web application designed to eliminate the stress and complexity of splitting restaurant bills. By providing an intuitive, user-friendly interface, the app transforms a potentially awkward social interaction into a seamless experience.

Key Features:

  • Automatic tax and tip calculations
  • Support for uneven bill splits
  • Item exclusion functionality
  • Real-time updates and adjustments

Technical Implementation

I crafted the application using a robust tech stack, focusing on creating a responsive and interactive user experience. The front-end was built with HTML5 and CSS3, ensuring a modern and adaptive design. JavaScript, along with jQuery, powered the dynamic calculations and form interactions, while Bootstrap provided a responsive layout framework.

Dine-In Wildcat: Campus Dining Companion ✨

Dine-In Wildcat is a transformative mobile application designed specifically for Northwestern students, bridging the gap between campus life and local culinary experiences. The app provides a comprehensive dining solution that goes beyond simple restaurant listings.

Innovative Features:

  • Personalized restaurant recommendations
  • Exclusive student discounts
  • Real-time menu updates
  • Cross-platform mobile support

Development Journey

Developed using React Native for cross-platform compatibility, the app leverages Firebase for backend services and real-time data management. I integrated multiple APIs to fetch restaurant data, manage user authentication, and provide up-to-date discount information, creating a seamless and engaging user experience.
